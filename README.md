# Health monitoring app

A Flutter project in our CMSC 23 - B5L class, this allows students and employees to make health entries each day.

### Dependencies:

The app requires the following dependencies:

- firebase_core
- provider
- cloud_firestore
- firebase_auth
- intl
- qr_code_scanner
- qr_flutter
